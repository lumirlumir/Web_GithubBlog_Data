## Markdown Personal Convention

- console 창에 입력하는 keyword 혹은, code 관련 내용들은 code block(\`\`)으로 표현.
- JavaScript, ECMAScript 등 CS관련 용어들은 bold & italic(\*\*\* \*\*\*)으로 표현.
- 강조할 내용들은 밑줄(`<u></u>`)로 표현
